# Languages

The native languages spoken by a player character are determined by the [character’s class](/srd/index.php/Character_Classes "Character Classes"). These typically include the common tongue and an [alignment](/srd/index.php/Alignment "Alignment") language. Characters with high INT may learn additional languages (see ***[Ability Scores](/srd/index.php/Ability_Scores "Ability Scores")***).

## The Common Tongue

The common tongue (sometimes simply called ***Common***) is a language which is widespread among intelligent species. All player character races—as well as many monsters—are able to speak Common.

In some settings, the [referee](/srd/index.php/The_Referee%E2%80%99s_Role "The Referee’s Role") may decide that different cultures in the campaign world use their own language, instead of Common. In this case, characters must choose to speak Common or a specific regional language.

## Alignment Languages

All intelligent beings know a secret, unwritten language of gestures, signs, and code words associated with their [alignment](/srd/index.php/Alignment "Alignment"). This secret language allows beings of the same alignment to communicate. Beings of another alignment will recognise when an alignment language is being used, but will not understand. It is not possible to learn another alignment language except by changing alignment, in which case the former language is forgotten.
